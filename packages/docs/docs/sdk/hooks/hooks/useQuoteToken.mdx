# useQuoteToken

`useQuoteToken` hook is used to get Token object from the quote token address.

## Parameters

- address: `string` - The address of the quote token.

## Returns

`Token` object with the following properties:

- address: `string` - The address of the quote token.
- symbol: `string` - The symbol of the quote token.
- name: `string` - The name of the quote token.
- decimals: `number` - The number of decimals of the quote token.
- camelCased: `boolean` - Whether the quote token contract uses camelCased function names or not.
- usdcPair?: `Pair` - The USDC Jediswap pair of the quote token.
  - address: `string` - The address of the pair.
  - reversed: `boolean` - Whether the pair is reversed or not.

## Usage

```ts
import { useQuoteToken } from 'hooks'

const quoteToken = useQuoteToken(quoteTokenAddress)
```
